自诚和自欺

——读马克思《青年在选择职业时的考虑》有感

1835年，青年时期的马克思在面临升学和就业的问题时，发表《青年在选择职业时的考虑》一文，将自己对社会、家庭、自身的思考汇入思想浪潮。虽年仅17岁，此篇文章已经透露出一些伟人思考的端倪：人做出的选择应当来自理性认同基础上产生的热情，遵从本心比虚无缥缈的物质财富更加能使一个人充盈余生；在做出人生重大选择（如选择职业时），要平衡自身、家庭、社会的关系，切忌头脑发热一时冲动、浮想联翩不计后果、沽名钓誉妄自菲薄、天马行空无视实践、眼高手低错误估计；他指出人是社会动物，重视社会价值和社会地位的事实，但不关注于标量化社会地位和社会声誉，而是遵从人类的本心：高尚与否。珍视自己职业的人社会地位必然是高尚的，忠诚于自己所选择职业的人社会行为也必将保持高尚。根据历史的经验教训，他得出：为人类福利而劳动的职业是历史经验所赞颂，是社会人民所认可的高尚职业。

马克思是全世界无产阶级的伟大导师，伟大的政治家、经济学家、哲学家、革命理论家。马克思一生的非凡成就，是从少年时期的深刻觉悟和伟大理想息息相关的。经由此文，我们可瞥见年少时期的马克思已成为伟人的前身，全文虽然没有明确提到自己所意向的职业，但在结尾：“历史承认那些为共同目标劳动而自己变得高尚的人是伟大人物。”已告诉读者，马克思想成为伟人，为社会、为人民的幸福而终身劳动。

历史在这一刻选定了他，神在这一刻肯定了他。

开篇点题，马克思通过动物与人的对比指出人的高尚，使用“神”这个颇为宗教意味的字词，但我想，这里的“神”指的是社会环境，而社会是人类种族组成的集体结构，可以是“人类”这个大种族的集体心声。对于每一位在社会中生活的人来说，“神”即自我，“神”即心声。青年人在走出学校象牙塔的保护下，迈出在社会中的第一步，便是职业——“最适合他、最能使他和社会都得到提高的地位”。失之毫厘谬以千里，第一步是打下人生地基的基本步，也是决定人生是否有价值、有意义的决定性一步。

人生重大之选择，不怕于做出抉择的时间点早晚，而怕于随波逐流盲目跟风、欲望驱使忽略初衷、碌碌无为忽略实际。当今社会逐渐拥挤的就业赛道，无意义内卷的高材应届生们，原子时代浮躁不平的个人生活……细分到个人二十几年逐渐趋于同一的学生生涯，机械系统化的教育行业与资本主义利益相互交织，功绩社会下家庭、社会、个人客观因素带给职业选择的容错率大大降低、经济下行与世界巨变，这个“好时代”似乎摇摇欲坠。然而，正如马克思在文中所言，“我们自己的理智不能给我们充当顾问”，与现实深度接轨、与时代深度融合的正是我们自己的经验、观察和实践。一代人有一代人的使命，一辈人有一辈人的坚守。青年人在做出职业选择之际，更需要关注家庭、体质（也即能力和天赋）、尊严（自诚和自欺），以及社会的多方面考虑，不是青年选择时代，而是时代铸就青年。

家庭方面，排除个别家庭，大部分家庭父母确实是在最接近自身环境下的上一个时代的“成功体”，青年人应该对父母的言传身教保持自我审视、秉承扬弃。正如妈妈辈的《甄嬛传》延续十余年经久不衰，也正如互联网甄学二创愈发壮大，二者在继承和发展中相互作用，螺旋上升式前进。

北国的秋冬

颇有感触的这几天。

昨天明明是是十几度的艳阳高照，今日早起，气温跌至两度，虽然仍然艳阳高照，但寒风刺骨，秋冬在此刻已深深交融。

最近的日推里多了许多首纯音乐，睡觉时也泡在耳机里，纯音乐带着我的灵魂出走飞往家所在的方向，无奈时间太短，路程太远，北风太浓，八周的日日夜夜，一无所获地睁眼，怅然若失。

昨天听到一首《给奶奶的歌》，落满收藏品和我童年快乐的地下室突然蹦到我的脑海，登上七层台阶，院子里的枇杷树，走道左边有棵桃树，走道右边也有棵桃树。池塘，我始终不敢跳过去的井盖对岸，颠簸的电动车在石子上颠簸着，妈妈问我“冷不冷”“困啦”，还让我把手放到她屁股底下。我爱吃枇杷爱吃桃，初中之前我都有枇杷吃，现在每个假期回去爷爷还是会给我桃吃。

师附正门的花开了，穿过正元和羊肉米线门口的板凳桌椅、姹紫嫣红的晚霞，小小的多那甜甜圈亭子在高三那年倒闭，可惜我还没有吃上一口，就像师大实验门口的多那甜甜圈，我假期回去的时候已经变成寿司店了。

我还记得小学的时候妈妈给我买mp3，我喜欢在里面下缪斯的歌，出去旅游到天南海北，我睡觉都在听snow halation。我不喜欢纯音乐，妈妈喜欢钢琴和纯音乐。

现在我为失眠的夜晚建一个纯音乐歌单，暂时收纳惶恐无助的自己。我是在享受琴键击打琴弦的干净纯粹，还是在追赶过去的我。我永远在追赶过去，在图书馆里不断哽咽最终用手挡住脸。

刚刚在刷牙的时候突然意识到，高中竟然变成一种奢望。被奴役的我啊，说到荣归故里，何时又出走过我的家乡？

人各有志，坚强或是独立，成长的苦痛把我强压在时代的五指山下。这么费劲，只是想拥有陪伴的底气。

我与史铁生与他的地坛

选择在期中周结束搓出这一篇文章的一般原因是因为我总是在无缘无故地瞎忙，另一半原因是因为我懒。早在一周前就已经读完《我与地坛》，并且在第十章光荣地颅内高潮，似乎很久都没有体会到读完一段文字之后激动的心颤抖的手，它引导着我读过中华上下五千年外国上下不知道多少年来文人骚客对一年四季不同创作背后的多重含义和解读，让我这个完全丈育的小东西（某种byd黑话）对为什么古今中外有这么多歌颂四季自然风光的作品产生了茅塞顿开醍醐灌顶的一丝理解和顿悟，当然还有更深层次的意义似乎还需要更多的引导和点拨。也许在读的各位看到以上这一段非常通顺白话的长句子读起来会很断气，但是这些确实是我为《我与地坛》写下的开头段读后感，用于表达我完全的膜拜和感谢之心，《我与地坛》真的是一本非常好的启蒙人生鸡汤书，不同于其他人把鸡汤端到你嘴边告诉你吃啊吃啊不教你怎么张嘴，这本书在用亲身演示不厌其烦地展示成千上万遍鸡汤到底应该怎么吃，从宇宙大爆炸物种诞生的宏观视角缩小到咱们一个21世纪人类小婴儿出生和会拿勺子开喝，你吃的心服口服，吃的理所应当，吃的心甘情愿，吃的五体投地。

以前的心境平和到不能理解大人说青春热血到底是什么意思，大学生太幼稚太理想太志气又是什么意思，摇滚不死河北石家庄又双叒是什么意思，怒放的到底为什么在春天，欲望为什么要叫春光乍泄。《我与地坛》第十章看了两遍，第二遍才隐约明白，春光迷乱。“灵魂一到人间便被囚入有限的躯体，那灵魂原本就是多少梦想的埋藏，那躯体原本就是多少欲望的储备！”人有三大欲望，情欲口欲和物欲，原来自己所有的思想都有迹可寻，原来自己所有的意识都被欲望完全支配，思到如此，我竟产生一丝愧疚和释然，活了十九年最近才明白，干嘛要当人以为多浪漫。

《我与地坛》在我这里最成功的地方在于，它真的做到了把冥冥之中感觉有关系但又说不出逻辑道理的两个东西用非常成功的个体主观逻辑串联起来，并且成功把我洗脑变成其信徒，把自己长久以来很想关联的不同东西串起一个网，告诉所有我原来平凡的人也有大智慧，史铁生何尝不是一种我的逻辑嘴替。感谢上帝，感谢耶稣。

最近期中周也是在意想不到的时候到来了，回想起上个学期的期中周那会儿，我的大物和扫雷设计让我痛不欲生了三天，堪堪以请求外援支持的结果堪堪结束。当然大物期中考得一坨，我好像记得好几道选择题我直接干错四题，本来以为人生无望结果五一过后的期末怒刷五百道，到现在也不知道自己大物到底是怎么学的，非常笨拙并且生疏没有方法论。这个学期逐渐忙碌了起来，课内课外都有些事等着我去完成，战线拉长，生活还是给我这个小可颂上压力了。有些时候明显自己变得健忘甚至愚蠢，经常要在出寝室门之后才发现自己又有东西没有拿，站在原地踌躇三秒，叹了口气嘀咕一声又是认命。题目看得也不太认真，发现了很多没有沉下心来急急躁躁就去完成的错误和后悔，这些都化成量化指标告诉我血的教训。写于现在的随笔，2023最后的七十天拜托麻烦请好好过。

物联网技术导论笔记

2023/09/28

  • ID barcode条形码（一维码）
    barcode reader识别 激光光源打到条码上会有反射 转化成电信号（光转化成电）翻译成不同字符
    基本包括起始字符和终止字符，有些还包括校检字符（没有具体含义，通过一定算法得出）

典型种类：
1.UPC-A 北美，一般零售商品场景；
2.UPC-E UPC-A精简版
3.codabar 图书馆
4.EAN-13 标识零售（全世界）
5.interleaved 2 of 5 五个黑条和五个白条连续交织在一块表示两个字符
6.code39 美国
7.code93 精简版
8.code128 能编码ASCII
9.bookland EAN 专门表示图书，映像，建材（ISBN）
barcode reader条形码识别器：光源，光学传感器，翻译光学信号变成电信号，通过解码器进行翻译（笔装、激光、摄像头、透镜全像）
条码系统：产生条形码，解码

  • 二维码：相对于一维码，编码表示的信息更多，使用空间更小

典型种类：
1.PDF-417 高度增加，所编码的信息增加，理论上无限制
2.Datamatrix数字矩阵码，大小可变
3.QR code（快速响应码）

  • IC card：嵌入式集成电路，认证身份，数据存储
    接触式IC卡/无接触式IC卡（无线射频识别，不需要内置电池）

典型应用：信用卡/身份识别/健康管理卡

  • RFID（最早物联网提出基于该技术，通过反射能量进行通信）射频识别
    无线非接触式系统，从标签里读取数据，存储数据，快速识别，不需要物理接触
    一般的RFID系统组成：RFID tag（附着在一般物体上）-与天线进行无线通信-通过网络传输到后端服务器，进行可视化

    • RFID标签：存储电信息，不要求读取器的光线；只读/可读可写
    • 标签组成模块：天线（面积比较大）、能量供给、无线发送接收模块、中心逻辑电路、信息存储模块（内存模块）
    • 标签的类型：
      1.主动（有电池，主动向外发送信息）/被动（无电池，不能主动向外发送信号，被动接受外界信号）/半主动（有电池，但不能主动向外发送信息）
      2.memory的类型进行分类：只读（出厂）/写一次读多次/可重复读写
      3.无线通信频段：Low-frequency（30-300khz），high-frequency（3-30Mhz），UHF（300MHZ-1Ghz），Microwave（2-30Ghz）

  • reader读取器：给标签进行供电、建立无线的数据通信、分析读取结果、与后端服务器进行通信、可以同时读多个标签；

  • 无线工作原理：reader发送无线电波，tag闭合线圈产生电场

  • RFID通信流程：服务器管理多个reader，reader和多个tag进行通信，reader发送采播信号，打到标签上，tag进行处理，发送给天线，天线发送返回给reader，reader进行解码，返回后台服务器

    • EPC码：可以表示到某一个商品
    • EPCglobal network：EPC借用该系统进行运作，管理动态信息
    • 由：对象命名服务（ONS），EPC发现服务（discovery），EPC信息服务（information）
    • ONS：发现这个商品的EPC码
    • 发现服务：搜索引擎，查找EPC码和商品信息
    • 信息服务：服务器，存储商品的详细信息
      生产厂商生产商品贴上tag，里面有EPC码，所有信息存储-到中心节点进行信息注册-零售点进行扫描，并在发现服务中进行注册（便于他人查找）-应用商查找
    • 应用：商品自动识别、野生动物追踪…
    • 特点：处理信息多，多线配合

分治思想处理最大子序列

核心思想就是二分法无限不断持续细分，用递归来实现，把每一个细分板块都看作独立的数列处理：

int Maxsequence(int array[],int left,int right){
    int center,i;
    int maxleftsum,maxrightsum;   //左半边最大子序列&右半边最大子序列
    int leftsum,mleftsum,rightsum,mrightsum;   //两边分别处理  
    if (left==right){    //基础情况：只有一个元素
        return array[left];
    }
    //先判断完成基础情况再进行递归
    center=(left+right)/2;     
    maxleftsum=Maxsequence(array,left,center);    //递归左半边
    maxrightsum=Maxsequence(array,center+1,right);  //递归右半边
    leftsum=0;mleftsum=0;  //别忘了初始化
    for (i=left;i<=center;i++){
        leftsum+=array[i];
        if (leftsum>mleftsum){
            mleftsum=leftsum;
        }
    }
    rightsum=0;mrightsum=0;
    for (i=center+1;i<=right;i++){
        rightsum+=array[i];
        if (rightsum>mrightsum){
            mrightsum=rightsum;
        }
    }
    return max3(maxleftsum,maxrightsum,mrightsum+mleftsum);   //左半边最大子序列、右半边最大子序列和中间最大子序列（发现规律得来）
    }

动态规划处理最大子序列

从开头开始，遇到负数就ban掉，和是正数就继续遍历，如果数列第一个元素是负数则重来（和为0）

int online-maxsequence(int array[],int n){
    int sum=0,maxsum=0;
    int i;
    for (i=0;i<n;i++){
        sum+=array[i];  
        if (sum>maxsum)
            maxsum=sum;
        else if (sum<0)   //换个角度和为负数则归零
        sum=0;
    }
    return maxsum;
}
